## how to use sanity with nextjs

how to use

```
npm i -g @sanity/cli
&&
sanity init
```

use sanity.json then use api in .env

## how to dev in this project

start web

```
npm run dev
```

start sanity api

```
cd santity
&&
npm start
```

### if you want deploy sanity

```
cd sanity
&&
sanity deploy
```

my api demo is deploy on https://test-123.sanity.studio/
